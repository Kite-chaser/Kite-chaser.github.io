gcd

用(a, b)表示gcd(a, b)

(a, b) = (b, a % b)

(a, b) = (b, a - b) (a > b)

exgcd

求解ax + by = gcd(a, b)


void exgcd(int a, int b, int &g, int &x, int &y)
{
	if (!b) { x = 1; y = 0; g = a; return; }
	exgcd(b, a % b, g, y, x), y -= (a / b) * x;
}

证明：

裴蜀定理

方程 ax + by = c 有解的充要条件是gcd(a, b)是c的因数

费马小定理

p为质数且a不是p的倍数时，ap−1=1(modp)a ^ {p - 1}= 1\pmod pap−1=1(modp)

欧拉定理

a和m互质时，aϕ(m)=1(modm)a ^ { \phi(m)} = 1 \pmod maϕ(m)=1(modm) 即ac=ac mod ϕ(m)(modm)a ^ c = a^{ c\ mod\ \phi(m)} \pmod mac=ac mod ϕ(m)(modm)

a和m不互质时，ac=ac mod ϕ(m)+ϕ(m)(modm)a ^ c = a^{ c\ mod\ \phi(m) + \phi(m)} \pmod mac=ac mod ϕ(m)+ϕ(m)(modm) (c > ϕ(m)\phi(m)ϕ(m))

线性筛

素数，欧拉函数，莫比乌斯函数，约数个数

求逆元

模数为质数 ksm(a, p - 2)

exgcd

线性求逆元 inv[i] = mod - (mod / i) * inv[mod % i]

推导

设t = mod / i， k = mod % i

-t * i == k (% mod)

-t * inv[k] = inv[i] (% mod)

inv[i] = mod - (mod / i) * inv[mod % i] (% mod)

中国剩余定理

int CRT()
{
    int ans = 0, M = 1, x, y;
    for (int i = 1; i <= k; i++) M *= m[i];
    for (int i = 1; i <= k; i++)
    {
        int tp = M / m[i];
        exgcd(tp,b[i],x,y);
        x = get_inv(tp, m[i])
        ans = (ans + tp * x * a[i]) % M;
    }
    return (ans + M) % M;
}
